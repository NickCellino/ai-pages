Demo 3: Qianfan OCR bridge (experimental)

Check Qianfan-OCR browser compatibility and optionally try remote inference endpoints.

Open Qianfan bridge demo →

How this works

  1. The page loads a TrOCR model through @xenova/transformers.
  2. Your image is kept in-browser and sent to the model pipeline.
  3. Predicted text is shown in the output panel.

Tip First run may take a bit while model files download and cache.
